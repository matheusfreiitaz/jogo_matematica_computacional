💰 PurpleFinance
PurpleFinance é uma plataforma moderna de controle financeiro pessoal que permite ao usuário acompanhar saldos, registrar transações e monitorar metas financeiras, tudo isso em uma interface elegante com tons de roxo.


✨ Funcionalidades
📊 Painel com visualização de saldo, renda e despesas

💸 Registro e listagem de transações (renda e despesas)

🎯 Metas financeiras com barra de progresso

📈 Gráficos interativos com Chart.js

🌙 Interface responsiva com design moderno usando Tailwind CSS

⚙️ Desenvolvido com Vue 3 Composition API

🚀 Tecnologias Utilizadas
Vue.js 3

Chart.js

Tailwind CSS

HTML5 & JavaScript
